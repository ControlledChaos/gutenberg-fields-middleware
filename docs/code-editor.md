# code-editor

#### value:

The source code to load into the code editor.

- Type: `string`
- Required: No

For more read gutenberg [readme](https://github.com/WordPress/gutenberg/tree/master/components/code-editor).

**Example:**

```js
editorContent: {
	type: 'string',
	field: {
		type: 'code-editor',
	},
}
```

### Return

This will return code-editor output.

- Type: `string`
